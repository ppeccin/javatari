javatari

A multiplayer Atari 2600 emulator written in pure Java with no external libs.

Download this project as a .zip file Download this project as a tar.gz file

Download the executable JAR version 1.06 here

Just open it! Needs Java 6 or greater and Atari ROMs.

Visit the project homepage for changelog and release notes.

To start the emulator in Multiplayer modes:


$ java -cp javatari.jar main.MultiplayerServer
or
$ java -cp javatari.jar main.MultiplayerClient

Features

  • Unique Client-Server multiplayer mode. Runs great in low-latency networks such as LANs.
  • Cheat and turn off Collisions. Finally discover the ending of River Raid!
  • Complete Save State/Load State functions.
  • Scanlines and TV screen emulation modes.
  • Real Atari console user interface.

Controls

P1 Controls: Arrow Keys / Space: Fire
P2 Controls: F,T,G,H / A: Fire
Alt + J: Swap P1<>P2 Controls
Alt + L: Connect/Disconnect Paddles

Alt + ENTER: Toggle FullScreen
Alt + R: Toggle CRT modes
Alt + Q: Toggle Filter
Alt + D: Toggle Debug modes
Alt + C: Toggle Collisions
Alt + V: Toggle NTSC/PAL
Alt + P: Toggle Pause
Alt + F: Next Frame (during Pause)

Ctr + 1-0 Save State
Alt + 1-0 Load State

Shft + Arrows: Screen Size
Backspace: Default Screen Size

TAB: Fast Speed
